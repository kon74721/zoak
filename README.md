## zoak ##

### Run ###

```bash
npm install
npm run build
npm start
```

Then connect to http://127.0.0.1:8080/

### Add data ###

1. Copy your [czml](https://github.com/AnalyticalGraphicsInc/czml-writer/wiki/CZML-Guide) file to ```data/```
2. Run ```make -C data/``` to rebuild the data/index.js index.
